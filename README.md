# Rush it Studio Blog Site

A responsive blog website design with modern UI/UX elements.

## Features
- Responsive design that works on all devices
- Interactive navigation with mobile menu
- Blog post cards with hover effects
- Case study section
- Email subscription form
- Smooth animations and transitions
- Modern color scheme and typography

## How to Use

1. **Clone or download** the project files
2. **Open `index.html`** in your web browser
3. **Customize content** by editing:
   - `index.html` for content and structure
   - `style.css` for styling and colors
   - `script.js` for interactive features
